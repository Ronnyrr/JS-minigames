## JS minigames

Node based application containing some javascript minigames.
ES6 classes using jQuery library. HTML lay-out contains SWIG templating.

## Games

1. Memory ~ /memory
2. Simon Says ~ /response

## Install

1. npm install
2. gulp --dev
3. http://localhost:4321

## Generate static

1. gulp
2. Generates /dist folder
3. Open html files
